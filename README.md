# Welcome!

This repository consists only of code for the landing page :D

To see code for:

- **Models:** we direct you to [powerNAP](https://github.com/generalusermodels/powernap): an implementation of our model that runs directly on your own computer use, labeling and training at the same time.

- **Data collection:** we direct you to [NAPsack](https://github.com/GeneralUserModels/napsack): a system that handles only data collection, records and aggregates your computer use — screenshots plus input events (click, keypress, scroll, cursor move). It also groups activity into event bursts and uses a VLM pipeline to generate human-readable captions describing what happened.
